NanoDNS

A lightweight DNS server designed for Docker Compose environments, allowing dynamic resolution of service names and custom DNS records.

Features

Easy Configuration

Configure DNS records using simple environment variables.

Quick Start

services:
  dns:
    image: ghcr.io/mguptahub/nanodns:latest
    environment:
      - DNS_PORT=5353
    ports:
      - "5353:5353/udp"
docker run -d --name dns \
  -e DNS_PORT=5353 \
  -p 5353:5353/udp \
  ghcr.io/mguptahub/nanodns:latest
